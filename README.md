# SEIR_Projects
**----Project_1-----**

Python web scraper using requests and BeautifulSoup to fetch a webpage, handle basic restrictions with custom headers, and extract the page title, full body text, and visible links.

Implements manual word frequency counting and generates a 64-bit polynomial rolling hash for each word. Builds a weighted Simhash fingerprint for each webpage and compares documents by calculating the number of matching bits between their 64-bit fingerprints.
